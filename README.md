# EQVII Site

Technical foundation for the EQVII Holdings static website.

## Stack

- Astro
- TypeScript
- Plain CSS with CSS custom properties
- Static output for Netlify

## Commands

```sh
npm install
npm run dev
npm run check
npm run build
npm run preview
```

## GitHub Remote

```sh
git init && git remote add origin https://github.com/Eqleus/eqvii-site.git
```
